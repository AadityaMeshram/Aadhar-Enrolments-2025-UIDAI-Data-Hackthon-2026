# Aadhar-Enrolments-2025-UIDAI-Data-Hackthon-2026
As part of the UIDAI Data Hackathon 2026, organized by the Unique Identification Authority of India (UIDAI) in association with the National Informatics Centre (NIC) and the Ministry of Electronics and Information Technology (MeitY), I worked on an end-to-end data analytics project using Aadhaar Enrolment open data.
